Support.

Choisissez le canal le plus sûr et ne partagez jamais de token, clé API, source, URL de média, info-hash, chemin local ou session Stremio.

Problème produit

Consultez les problèmes existants ou ouvrez un ticket public sur GitHub Issues. Décrivez le comportement sans nommer le contenu regardé si ce détail n’est pas indispensable.

Incident de sécurité

N’ouvrez pas de ticket public. Utilisez le formulaire privé GitHub Security Advisory. Indiquez seulement les étapes minimales de reproduction et expurgez tout secret réel.

Supprimer un compte

Dans VAYRA, ouvrez Réglages → Compte → Supprimer le compte VAYRA. La suppression du compte distant entraîne la suppression en cascade du profil, des relations, groupes, invitations et rooms. Elle ne supprime pas automatiquement les données locales ni un compte Stremio indépendant.

Données locales

Utilisez les commandes de réinitialisation et de suppression de VAYRA sur l’appareil, puis désinstallez l’application si nécessaire. Une sauvegarde exportée reste sous votre contrôle et doit être supprimée séparément.

Demande liée aux données

Écrivez à privacy@eybo.tech depuis l’adresse du compte concerné. N’envoyez aucun mot de passe ni magic link.
